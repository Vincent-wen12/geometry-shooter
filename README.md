# 🎮 几何枪战 - 2D 多人射击游戏

一款基于 HTML5 Canvas 的多人联机几何射击游戏！

## ✨ 功能特点

- 🎯 **几何图形角色** - 六边形玩家 + 三角形敌人
- 🎨 **多彩配色** - 随机颜色分配，炫酷发光效果
- 🔫 **射击系统** - 鼠标瞄准，左键射击，装弹系统
- 👾 **AI 敌人** - 自动追踪玩家的三角形敌人
- 💥 **粒子特效** - 射击、击杀都有炫酷特效
- 📊 **HUD 界面** - 生命值、弹药数、击杀数显示
- 🏆 **击杀信息** - 实时显示击杀信息
- 🌐 **多人联机** - WebSocket 实时同步
- 🏠 **房间系统** - 创建/加入房间，好友一起玩

## 🚀 快速开始

### 1. 安装依赖

确保已安装 Node.js，然后在项目目录运行：

```bash
npm install
```

### 2. 启动服务器

```bash
npm start
```

### 3. 开始游戏

在浏览器中打开：`http://localhost:3000`

## 🎮 游戏操作

| 按键 | 功能 |
|------|------|
| W / ↑ | 向上移动 |
| S / ↓ | 向下移动 |
| A / ← | 向左移动 |
| D / → | 向右移动 |
| 鼠标 | 瞄准 |
| 左键 | 射击 |

## 🎯 游戏玩法

### 单人模式

- 不输入房间ID，直接点击"开始游戏"
- 敌人会从四面八方不断出现
- 击杀敌人获得分数
- 被敌人碰到会掉血，生命值归零游戏结束

### 多人联机

1. 第一个玩家输入任意房间ID（比如 `abc123`），点击开始
2. 其他玩家输入相同的房间ID，点击开始
3. 进入同一个房间后可以看到彼此
4. 可以一起合作对抗敌人！

## 🎨 游戏特色

### 视觉效果
- 深色科幻风格背景
- 网格地面
- 角色发光特效
- 粒子爆炸效果
- 子弹拖尾效果

### 游戏机制
- 30发弹药，耗尽自动装弹（1.5秒）
- 敌人随机生成，速度和大小不一
- 碰撞检测和物理移动
- 实时玩家列表和击杀信息
